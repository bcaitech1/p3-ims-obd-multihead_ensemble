# Semantic Segmentation for Sorting Recycled Items
## Usage
### Train
```
python train.py
```
- You can use args like `python train.py --batch_size 32`

### Test (Inference)
```
python inference.py
```
- You can use args like `python inference.py --batch_size 64`
